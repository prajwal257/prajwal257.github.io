YouStream
Page Design

Description

YouStream at its core is video streaming website, that was builted keeping Youtube as a main inspiration. The website has many of the features that are present in youtube like, from most basic watching videos and being able to see the description of the publisher of that video, being able to like share the url so that other people connected to network (incase of youtube its 'Internet') be able to see it as well. Features ranging from storing the watch progress for individual user and individual video to being able to publish the content of own. The content can be published and every element video i.e. thumbnail, desciption can be edited as well.

The web application was made by using PHP was used for backend programming as a server-side language. And MySQL for database functionality. For the frontend HTML, CSS, JavaScript was used.
